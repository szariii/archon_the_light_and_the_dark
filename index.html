Archon

ARCHON THE LIGHT AND THE DARK

Light Side

Arrow up - move up

Arrow down - move down

Arrow left - move left

Arrow right - move right

Slash (/) - make action / attack

Dark Side

W - move up

S - move down

A - move left

D - move right

C - make action / attack

Menu

Arrow up - change players

Arrow down - change starting side

Slash (/) - start game
